아파치(정적) + 톰캣(동적) = 아파치 톰캣(정적, 동적 페이지 처리하는 서버)

클라이언트 -> 요청 -> 서버(Controller -> Service -> DAO -> DB)

-------------------- 설치 및 세팅 --------------------
https://tomcat.apache.org/
왼쪽 메뉴 Download - Tomcat 10 - Core : zip 클릭

이클립스 - Window - Preferences - Server 탭 열고 - Runtime Environments - Add - Apache 탭 열고 - Apache Tomcat v10.1 선택 후 - next 클릭 - browse 클릭 - 설치한 아파치톰캣 폴더 찾아서 선택 - Finish - 적용
이클립스 - 하단 Servers 창 클릭 - No servers are available. Click this link to create a new server 클릭 - Apache 탭 열고 - Tomcat v10.1 Server 선택 - server name 에 Servlet Server
이클립스 - 하단 Servers 창 - 내가 만든 서버 더블 클릭 - 열린 창의 하단 중 Overview는 서버에 관련된 설정, Modules은 사용한 포트번호에 관련된 이력이 기록되는 공간 remove를 통해 지울 수도 있음
이클립스 - Window - Showview - Other - Package Explorer 검색 후 Open - 이클립스 하단에 열린 Package Explorer 왼쪽 탭으로 옮김

JSTL 설치
구글 드라이브 lib 폴더 안에 JSTL.zip 파일 받고 압출 풀어서 이클립스 해당 프로젝트 안 lib 폴더에 집어 넣기

이클립스에서 lombok 인식시키기
- lombok.jar 파일 이클립스 실행 폴더에 복사해넣기 - Powershell 실핸하여 방금 복사한 lombok.jar 파일이 있는 경로로 이동하여 java -jar .\lombok.jar 입력하기
- 이클립스 실행 파일 알아서 찾았다면 Install/Update 버튼 누르기 - 파일 못 찾았다면 - Specify location 버튼 누르고 - 이클립스 실행 exe 파일 찾아서 선택하고 Install/Update 버튼 누르기

-------------------- 기타 -------------------- 
xml 파일안에 빨간색 X 표시 없애는 방법 : Window - Preferences - XML (Wild Web Developer) - Download extenal resources like referenced DTD, XSD 체크
Dynamic Web Project 생성 시 - 이름 설정 - Apache Tomcat v10.1, modul 6.0 인지 확인 - next - next - Content root "/" - Generate web.xml deployment descriptor 체크박스 선택 후 - Finish
이클립스 html, css, xml 파일 만들 때 Encoding 기본값 설정 변경하는 법 - Window - Preferences - 검색창에 각각 html, css, xml 등 검색 후 - ~~.file 이란 탭에 들어가 Encoding 설정을 UTF-8로 설정 변경
